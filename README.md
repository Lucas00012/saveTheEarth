# saveTheEarth

Jogo estilo "Space Invaders" desenvolvido em HTML, CSS e javascript

* Não foi utilizado nenhuma biblioteca

* Feito com JS puro

# Testando o projeto

* Abra o arquivo html

* Quando estiver pronto, basta clicar no botão 'Jogar'
